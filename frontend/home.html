About the project

Our project is a variation on a smaller scale, but easier to understand, of WireShark.

The app consists of two main components: the tutorial part and the main panel. In the main panel, users can interact with a form to analyze network traffic. The form includes the following fields: platform: Users can select the platform they use to connect to the Internet, such as WI-FI or Ethernet, Operating System: Users can choose the type of operating system they are using, such as Windows or Linux.

By providing this information and submitting the form, users will be able to analyze the network traffic based on their platform and operating system, using the data from the inserted PCAP file. This allows for a comprehensive understanding of network activity and facilitates troubleshooting and analysis tasks.,/

What is WireShark?

Wireshark is a free and open-source packet analyzer. It is used for network troubleshooting, analysis, software and communications protocol development, and education.

What is a PCAP file?

A PCAP (Packet Capture) file is a common file format used to store network packet data captured during network monitoring or analysis. PCAP files are typically created by network sniffing tools or packet capture software, such as Wireshark or tcpdump. A PCAP file contains a record of network packets captured from a network interface or a specific network segment. Each packet within the file includes various details such as source and destination IP addresses, port numbers, protocol information, packet length, and the actual packet data. Note: our app support only PCAP files, if you upload other types of files you will receive an error message.

What do you have to do?

You will have to write in the input label the absolute path to a PCAP file, then press the "Save" button.

We offer you the opportunity to see only part of the data that we read from the file provided by you, through the select under the "Save" button.

What will we do with the PCAP file?

We will not keep the file or other user data.
